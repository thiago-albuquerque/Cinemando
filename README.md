# Cinemando

Este projeto é um site de informações sobre filmes utilizando a API The Movie Database (TMDB).

## Tecnologias utilizadas:
+ HTML
+ CSS
+ Javascript
+ ReactJs
+ API The Movie DB (TMDB)
